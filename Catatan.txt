(Install express) :
pertama :
1. buka command prompt
2. mkdir namaFolder (--> example : mkdir express-api)
3. cd namaFolder (--> example : cd express-api)
4. npm init --yes
5. npm i express --save (--> untuk install express)
6. buat file server.js
7. buat file .gitignore (ignore.js)

(install nodemon)
8. npm install -g nodemon (--> untuk install nodemon)

(install body parser)
9. npm i body-parser

(install mongoose)
10. npm install mongoose (--> untuk install mongoose)

(Run aplikasi ) :
1. node server.js atau nodemon server.js
2. aplikasi bisa dibuka di localhost:3000 (-->localhost sesuai yang dibuat)
